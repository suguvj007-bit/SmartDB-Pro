# SmartDB Pro – Custom Database Management System

## Overview

SmartDB Pro is a custom Database Management System developed using Python Data Structures and Django. The system allows users to efficiently manage student records through a user-friendly web interface.

The project demonstrates the practical implementation of Data Structures, Search Algorithms, CRUD Operations, Activity Tracking, and Analytics within a Django-based application.

---

## Features

### Dashboard

* View total records
* View total searches
* Recent activity tracking
* Quick navigation to all modules

### Records Management

* Add new records
* View all records
* Edit existing records
* Delete records

### Search Center

* Fast record search using indexing
* Search count tracking

### Analytics Dashboard

* Total record statistics
* Course-wise student distribution
* Interactive charts using Chart.js

### Activity Logs

* Track user activities
* Display recent actions
